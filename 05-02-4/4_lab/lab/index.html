    Задача 1. Факториал

    Используя рекурсию, реализовать функцию вычисления факториала числа.

    Задача 2. Многоуровневый список

    Дан массив вида, который может иметь неограниченную вложенность.

    $someArray = [
   'id'    => 1,
   'name'  => 'item1',
   'items' => [
      [
         'id'    => 2,
         'name'  => 'item2',
         'items' => [],
      ],
      [
         'id'    => 3,
         'name'  => 'item3',
         'items' => [],
      ],
      [
         'id'    => 4,
         'name'  => 'item4',
         'items' => [
            [
               'id'    => 5,
               'name'  => 'item5',
               'items' => [],
            ],
            [
               'id'    => 6,
               'name'  => 'item6',
               'items' => [],
            ],
         ],
      ],
   ]
]

    Требуется реализовать рекурсивную функцию, которая, на основе данного массива формировала список. Для формирования списка используются теги <ul></ul><li></li>
